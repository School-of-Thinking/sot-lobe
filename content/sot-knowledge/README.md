## What This Is

A clean, structured knowledge package for deploying SOT (School of Thinking) cognitive methodology to any AI agent, company, or integration. Contains no personal correspondence, no internal session logs, and no operational metadata.

## Files

| File | Contents |
|------|----------|
| `framework.md` | Core SOT concepts: cvs2bvs, GBB, x10, cAIos, Intelligence Trap, Value Fountains |
| `neuroscience.md` | Brain science underpinning SOT: PFC, limbic system, PRR, curiosity circuit |
| `curriculum.md` | Full SOT curriculum: TBD (30 lessons), ATLC (20 lessons), AIXIO (10 lessons), WOMBAT Theory |
| `thinking-tools.md` | All 12 SOT thinking tools — purpose, format, when to use |
| `coaching-style.md` | QRH Styleware, GBB Leader Style, coaching protocols, assessment rubrics |

## How to Use

1. **For agent injection**: Load all 5 files as system context. Start with `framework.md`, then `coaching-style.md`.
2. **For skill packaging**: Bundle as a `.skill` archive using the OpenClaw skill format.
3. **For LLM fine-tuning**: Use `curriculum.md` for example DFQ responses; use `coaching-style.md` for tone calibration.
4. **For commercial deployment**: See `coaching-style.md → Commercial Application` for ROI figures and pitch structure.
